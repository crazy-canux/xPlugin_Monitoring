================================================================================
Nagios Plugin - JIT's Application
================================================================================

:authors: Monitoring & Reporting,
          CC Industrialization & JIT Applications

This is the API reference for **JIT** application's plugins for Nagios. These
plugins use what is called the `XML interface`_, gateway between Monitoring &
industrialization applications.

This documents the `XML interface`_ classes in order to use or create new plugins
using it.
